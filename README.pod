=head1 INSTALLATION

Sub::Way installation is straightforward. If your CPAN shell is set up,
you should just be able to do

    % cpan Sub::Way

Download it, unpack it, then build it as per the usual:

    % perl Makefile.PL
    % make && make test

Then install it:

    % make install
